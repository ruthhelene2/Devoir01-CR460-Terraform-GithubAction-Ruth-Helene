# Devoir01-CR460-Terraform-GithubAction-Ruth-Helene
Dépôt créé dans le cadre du cours CR460 – Infonuagique. Ce projet met en place un pipeline CI/CD utilisant GitHub, Terraform et Microsoft Azure afin d’automatiser le déploiement de ressources cloud telles qu’un groupe de ressources, un réseau virtuel, une machine virtuelle et un conteneur Docker.
